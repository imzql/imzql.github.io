记录一些常用的docker命令与项目

本文最后更新于：2 个月前

最近喜欢上了docker部署，但是有些命令去文档看

官网老慢了，一些常用的指令记录下来，用来备忘

常用指令

一键查看正在运行容器

docker ps

设置容器执行服务器开启自启

sudo docker update 容器的id --restart=always

优秀的docker项目

rsshub

首先拉取镜像
docker pull diygod/rsshub
部署应用，这里部署在公网ip的1200端口，需要提前开启防火墙
docker run -d --name rsshub -p 1200:1200 diygod/rsshub
进行宝塔的域名反代，即可
